Increasing inequality?

The Waldron analysis presents the following table. If we simulate two heterogeneous groups, one which always has twice the baseline mortality level as the other, then we will see the observed hazard ratio converge with age as mortality selectively removes the most frail individuals. If we let the baseline levels of both groups decline steadily, simulating the process of mortality improvement, then this will make it appar that the hazard ratio is rising over time.
